## Comandos de la api para manejar la Task List

### GET

| url            | Descripción        |
| -------------- | ------------------ |
| /api/taskslist | Entrega las tareas |

### POST

| url       | Descripción                                                                                            |
| --------- | ------------------------------------------------------------------------------------------------------ |
| /api/task | Añade una nueva tarea que tenga en [**key** - `contain`] y en [**value** - `El contenido de la tarea`] |
